🎓 超星/学习通刷课脚本使用教程

核心原理：利用GitHub Action云端运行Python脚本，模拟人工逐秒观看视频，无需本地挂机，降低检测风险

一用前必看

脚本依托GitHub Action云端运行，配置完成后可自动执行

触发方式有两种：

  • 修改仓库任意文件（首次提交脚本时触发）
  • 手动开启后，每天早8点定时刷课，每日自动观看6小时后关闭

二准备工作

  • 注册GitHub账户
    地址：github.com
    （访问需自行解决代理问题）
  • Fork目标项目
    地址：lispringing/SuperStar
    （建议点亮星标 ⭐）
    →Fork后自动跳转至个人仓库

    • 三配置与运行步骤

    1. 开启Actions功能

    进入个人仓库 → 点击「Actions」按钮 → 开启该功能

    手机端为折叠状态，需展开查看

    2. 编辑配置文件

    • 进入仓库目录 .github/workflows
    • 打开 main.yml 文件并点击编辑

    核心修改最后一行，替换为个人信息：

    # 修改前 - name: Run main.py run: python main.py -u 学习通账号 -p 学习通密码 -l 课程id # 修改后（示例） - name: Run main.py run: python main.py -u "your_phone_number" -p "your_password" -l "course_id"

    3. 提交并运行

    修改完成后提交文件，若账号、密码、课程ID无误，GitHub Actions将自动启动脚本，可在「Actions」页面查看运行状态。

    四开启每日定时刷课 更新于2024.12.6

  • 再次编辑 main.yml 文件
  • 找到定时配置段，删除原有注释（注意缩进格式）

    • 原内容：

    #是否开始每天8点定时刷课，若要开启请把下面两行注释去掉 #schedule: # - cron: "0 8 * * *"

    修改后：

    #是否开始每天8点定时刷课，若要开启请把下面两行注释去掉 schedule: - cron: "0 8 * * *"

    五课程ID获取方法

    • 电脑端打开超星学习通网页版，登录后进入目标课程
    • 浏览器地址栏中的第一串数字，即为该课程的ID
